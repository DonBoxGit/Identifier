Для корректной работы парсера, в мониторе порта Arduin IDE выбрать - "нет конца строки".
Команды управления из Serial
1. Команда: r(Regime) - Режим работы
	Параметры: 3, 4
	   Пример: r3 - вкл. режим №3
		   r4 - вкл. режим №4

2. Команда: s(Switch)
	Параметр: 0, 1
	  Пример: s0 - переключить управление яркостью дисплея с помощью Serial
		  s1 - переключить управление яркостью дисплея с помощью потенциометра R10

3. Команда l(Light) - Яркость дисплея
	Параметр: 0 - 100(величина в процентах %)
	  Пример: l80 - установить яркость дисплея 80%
		  l100 - устаность яркость дисплея 100%
